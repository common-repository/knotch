Adds the ability to embed Knotch widgets in posts.

== Installation ==

If you are hosting your own Wordpress install, go to the "Plugins" menu in your dashboard and search for the Knotch plugin, and simply install from there. Otherwise, create a new folder called "knotch" under /wp-content/plugins and unzip all contents into there. Once complete:

1. Activate the plugin by going into the "Plugins" menu in your dashboard and clicking "Activate" under Knotch.
2. Go to the new Knotch settings menu in the Wordpress dashboard and enter your credentials.
3. You should now be able to set up Knotch widgets while editing posts by enabling the menu under "Screen Options".

== Description ==

This plugin adds the ability to embed Knotch widgets at the end of your posts.

Usage requires a preapproved API key and secret that may be obtained by emailing insight@knotch.it.
